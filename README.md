# CI/CD 
Материал к вебинару про CI/CD для студентов школы Hexlet.


Для установки зависимостей проекта выполнить:
```
make install
```

Для запуска линтера:
```
make lint
```

Для запуска тестов с отчетом о покрытии:
```
make cover
```
